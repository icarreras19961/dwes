The admin bar

Rooms

Select

Is the one that will show you all the rooms of the hotel.

Insert

This allows you to insert new rooms to the database.

You just have to select the category and press submit to insert it.

Update

To update the values of your rooms you have to select the room id and the category of it.

Delete

To delete a room you will have to put the id of the room.

Customers

Select

Here you can see all the customers of the web page and a little information about each one.

As you can see on the picture it's a button to update every user of the database.

Insert

This page allows you to insert a new user in the database filling the inputs with the information of the user and pressing submit.

Update

In here you can change any fill of the user and presing submit y will change in the database.

Delete

As in the rooms delete you just have to enter the id of the client that you want to delete and press submit to delete it.

Reservations

Select

In this case the select shows a table of all the reservations on the database.

Insert

To do a reservation you have first of all to insert the date in, date out and smash the submit.

Then you will find all the available rooms to choose which one you want then press submit and it will redirect you to the main page with the reservation complete.

Update

To update a reservation you can insert th eid of the reservation and press submit.

Ones submit is clicked it redirects to the update form where you can change all the info of the reservation.

Also in the select you have the options in the same table that you can update the reservation, if you click it it will redirects you to the update form.

Delete

The delete works the same as the update. It has a button on the same options table, if you press it it will destroy the reservation.

MyReservations

This is like the selection of the reservation with the peculiarity of filtering the reservation from users that means that it will show you the reservations of the user that its logged on the webpage.
